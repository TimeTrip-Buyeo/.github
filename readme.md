# ⏳ TimeTrip

### 1400년 전 백제로 떠나는 시간여행  
부여 AR 역사관광 플랫폼

<br/>

<img src="./github readme.png" width="100%" />

<br/><br/>

## ✨ 프로젝트 소개

**TimeTrip**은  
부여의 백제 유적지를 기반으로 한  
AR 역사 체험 관광 플랫폼입니다.

단순한 유적 관람을 넘어,  
사용자가 직접 1400년 전 백제 속으로 들어가  
역사를 “체험”하도록 설계했습니다.

<br/>

---

## 🎯 핵심 기능

### 📍 AR 타임슬립 카메라
현재 유적지 위에  
1400년 전 백제의 모습을 AR로 오버레이합니다.

### 🎧 4개국어 오디오 가이드
한국어 / 영어 / 중국어 / 일본어 지원

### 🏺 유물 · 인물 컬렉션
유적지 방문 시 관련 유물과 인물을 수집하고  
컬렉션을 완성할 수 있습니다.

### 📸 부여 백제네컷
AR 셀카 및 SNS 공유 기능 제공

### 📖 월별 스토리 시스템
방문 월에 따라  
다른 역사 시나리오가 자동으로 제공됩니다.

<br/>

## 🏛 프로젝트 배경

부여는 유네스코 세계문화유산 유적 5개소가 밀집한 도시이지만,  
기존 관광은 정적인 관람 중심이라는 한계가 있었습니다.

TimeTrip은:

- 백제 멸망의 드라마틱한 서사
- 공공 3D 문화유산 데이터
- AR 몰입형 체험
- MZ 세대의 기록/공유 문화

를 결합하여  
“다시 방문하고 싶은 체류형 관광 경험”을 만드는 것을 목표로 합니다.
